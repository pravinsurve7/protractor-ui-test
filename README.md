# Protractor UI Automation

UI automation on website using protractor tool. Framework used 'jasmine'.
Headless feature available, check '_protractor.config.js_'

## Installation

```npm
npm install
```

## Run tests
Start the selenium server
```npm
npm start
```
Open new terminal and run
```npm
npm test
```
